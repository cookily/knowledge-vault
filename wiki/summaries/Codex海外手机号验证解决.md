# Codex 海外手机号验证解决方法

## 一句话总结
开启 ChatGPT 账号的 2FA 双重验证（Google/Microsoft Authenticator），可显著减少 Codex 登录时的海外手机号验证弹窗。

## 核心内容
- **问题**：Codex 登录时疯狂弹海外手机号验证，换节点/清 Cookie/接码平台都不稳定
- **解决方案**：ChatGPT 网页版 → 设置 → 安全 → 开启双重验证 → 绑定 Google/Microsoft Authenticator
- **原理**：OpenAI 风控采用"账号信用体系"，2FA 是强可信信号，会降低手机号验证优先级
- **关键因素**：长期开启 2FA + 稳定设备 + 稳定地区 + 正常使用习惯 = 更高账号权重
- **比接码平台更稳**：近期接码平台风控频率高、号段被标记、成本上升
