# 摘要：Claude Code 神级开发者 9 个 Skills 推荐

## 核心观点

ClawHub 上 Skills 已超过 5 万，但「装得多不等于提效率」。作者按三类分类推荐：**开发提效、信息获取、进化 & 蒸馏**。

## Skills 分类

### 🛠️ 开发提效类

| Skill | 功能 | 下载量 |
|-------|------|--------|
| **github** | 直接操作 GitHub（PR/Issue/CI），最常用 | 21.6K |
| **code-review** | 按维度（功能/风格/安全/性能）结构化 review | 社区最火 |
| **waza** | 轻量规范集合（代码风格/提交信息/文档模板） | - |

### 🔍 信息获取类

| Skill | 功能 |
|-------|------|
| **multi-search-engine** | 多引擎聚合搜索，技术调研必备 |
| **agent-browser** | Playwright 驱动，可点浏览器、填表单、截图 |

### 🚀 进化 & 蒸馏类

| Skill | 功能 |
|-------|------|
| **self-improving-agent** | 从纠正反馈中学习，用得越久越懂你 |
| **proactive-agent** | 定时触发 + 自动流程 |
| **dot-skill** | 把同事/名人/伴侣蒸馏成 AI Skill |

## 推荐安装顺序（四个打底）

1. **github** - 一上来就装，PR/Issue/CI 全在对话里处理
2. **self-improving-agent** - 长期收益最高，让 AI 学你的项目
3. **code-review** - 每次 commit 前兜底
4. **multi-search-engine** - 技术调研信息更全面

## 核心原则

> **先装能解决你每天痛点的，再装看起来有趣的。**
