# Hermes Agent 日报 5月21日-25日

## 一句话总结
Hermes 桌面版开源（Windows/macOS）、Maple-OS 多 Agent 协作中枢上线、Hermes 结合 MT5 实盘量化（46%收益/70%胜率）、长会话 Token 超 12 万的优化方案。

## 核心内容

### 5月21日（20条）
- **Opencode GO 订阅**：首月 5 美元起接入 DeepSeek，比直接用 Kimi 等更划算
- **Hermes + MT5 实盘量化**：群友实盘 1800 笔交易，多空胜率约 70%，累计收益约 46%
- **Grill Me 工作流**：先用"Grill Me"技能让 Agent 追问需求细节，再编程，减少返工
- **Scrapling 反爬库**：解决 Agent 自动化操作网页的反 BOT 验证

### 5月22日（16条）
- **Hermes 桌面版开源**：中文社区独立开发，完整 Hermes 内核，支持 Windows/macOS，代码已开源
- **Caveman Skill**：极简 AI 交互设计的开源 Skill
- **Helio AI 同事工作台**：可视化编排 + 多智能体协同运行
- **AgentKey 数据聚合**：一键接入微博/微信/小红书/B站/知乎/抖音数据

### 5月25日（31条）
- **Maple-OS 多 Agent 协作中枢**：统一工作台实现全链路多 Agent 工作流编排，已开源
- **hermes-browser-bridge**：Hermes 通过无头浏览器进行网页自动化操作，复用用户标签页
- **长会话 Token 优化**：单次请求超 12 万 Token → 阶段完成后开新会话，先让 Agent 总结关键经验为文档
- **启动变慢排查**：Skills 文件 IO 可能是瓶颈，建议打点定位耗时模块
- **DeepSeek 2.5 折**：限至 5 月 31 日
