# 摘要：WeFlow — 本地微信实时聊天记录查看与分析工具

## 一句话总结

WeFlow（github.com/hicccc77/WeFlow）是完全本地的微信实时聊天记录查看、分析与导出工具，支持微信 4.0+，功能覆盖消息防撤回、年度报告、群聊画像、朋友圈解密、多格式导出，并提供 HTTP API 供开发者集成。

## 核心内容

### 功能矩阵

| 功能模块 | 说明 |
|----------|------|
| **聊天** | 解密图片/视频/实况；支持修改、删除本地消息；实时刷新无需中间数据库 |
| **消息防撤回** | 阻止他人撤回已发消息 |
| **实时弹窗通知** | 新消息桌面弹窗，支持黑白名单 |
| **私聊分析** | 消息数量统计、类型分析、时段分布 |
| **群聊分析** | 群成员详情、发言排行、活跃时段、媒体内容 |
| **年度报告** | 按年/跨年度统计报告 |
| **双人报告** | 指定好友生成专属聊天分析 |
| **消息导出** | JSON / HTML / TXT / Excel / CSV / PGSQL / ChatLab 格式 |
| **朋友圈** | 解密图片/视频/实况；导出内容；拦截删除/隐藏；突破时间限制 |
| **联系人** | 导出好友/群聊/公众号信息；尝试找回已删好友（实验性） |
| **HTTP API** | 本地 5031 端口，支持原始 JSON 和 ChatLab 标准格式 |

### 平台支持

| 平台 | 架构 | 安装包 |
|------|------|--------|
| Windows 10+ | x64 | .exe |
| macOS | Apple Silicon (M 系列) | .dmg |
| Linux | x64 | .AppImage / .tar.gz |

### 与 ChatLog 的差异

| | WeFlow | ChatLog |
|--|--------|---------|
| 界面 | 图形化桌面应用 | Terminal UI |
| 版本要求 | 微信 4.0+ | 微信 3.x/4.0 |
| 特色功能 | 朋友圈解密、年度报告、消息防撤回 | 企业微信支持、MCP SSE 协议 |
| 导出格式 | 7+ 种 | 多种 |
| 开发语言 | Node.js (Electron) | Go |

### 注意事项

- 仅支持微信 **4.0 及以上**版本
- HTTP API 仍在早期阶段，接口可能变动
- **请负责任地使用，遵守相关法律法规**
