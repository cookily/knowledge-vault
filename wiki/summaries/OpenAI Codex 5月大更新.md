# OpenAI Codex 大更新：Appshots + /goal毕业 + 锁屏打工

## 一句话总结
Codex 迎来五大升级：Appshots 双击 Command 读取全屏+隐藏文本、/goal 长程自主编码正式毕业、Mac 锁屏后 AI 仍可远程编程、ChatGPT 插件攻入 PowerPoint、团队 Analytics 全面升级。

## 核心内容

### 1. Appshots（全新功能）
- **操作**：Mac 上双击 Command 键，当前窗口"贴"到 Codex 对话
- **核心能力**：不仅能读屏幕显示的内容，还能读取**未滚动到的隐藏文本**（文件路径、URL 等）
- **智能追加**：60 秒内连续截图自动追加到同一线程
- 平台：Mac 所有版本计划已上线

### 2. /goal 正式毕业
- 从实验阶段转正，所有 Codex 开发者可用
- 设定一个里程碑目标，AI 持续推进数小时甚至数天不停歇
- 可随时查看进度、调整方向或暂停
- 早上设目标 → 晚上回来发现已完成 80%

### 3. Mac 锁屏远程编程（"黑魔法"）
- 开启 "Locked Use" 设置后，Mac 锁屏/休眠状态下 Codex 仍可操控
- 从手机端安全远程操作 Mac 应用
- 配合移动端预览，实现 7×24 不下线

### 4. ChatGPT for PowerPoint
- 一句话生成整套可编辑 PPT
- 可从 Gmail/Outlook/SharePoint 拉取实时数据
- 全球所有 ChatGPT 用户层级开放 Beta

### 5. 其他升级
- **高级标注模式**：直接在网页/UI 上修改元素并实时预览
- **插件共享**：Business 用户团队内部分发自定义插件
- **Analytics 大升级**：活跃用户/Credits/Token/代码行数/插件使用量等多维度

### 数据
- 周活跃开发者突破 **400 万**（两周前 300 万）
- Business/Enterprise 用户数今年暴增 **6 倍**
- **50% 用户**用 Codex 干的事已经不是写代码
