# Hermes Agent 日报 5月26日-27日

## 一句话总结
Neo4j 知识图谱用于 Agent 记忆、Holographic 纯本地记忆系统、EchoMind v1.1.0 发布（永久记忆+防幻觉）、字节开源 Eino Agent 框架、Claude 自动化 Issue→PR→Review 闭环。

## 核心内容

### 5月26日（16条）
- **Neo4j 知识图谱**：替代 Obsidian 用于 Agent 外置记忆，AI 可编写脚本操作图谱
- **Holographic 记忆系统**：SQLite + FTS5 全文搜索 + 信任评分 + HRR，纯本地无需云端
- **Workbuddy 辅助排查**：将 Hermes 错误日志发给 workbuddy 获取修复建议
- **DeepSeek Pro 模型**：文档与条款分析场景推荐

### 5月27日（23条）
- **EchoMind v1.1.0**：永久记忆+自我进化引擎+七类记忆+Few-Shot锚定+防幻觉污染
- **字节 Eino 框架**：Go 语言，运行时开销极低，企业级开箱即用
- **Claude 自动化 PR 流水线**：Issue→自动评估→生成代码→提交 PR→AI Review 闭环
- **AI 前端工作流**：GPT Image 2 生成 UI 设计图 → Claude/DeepSeek V4 Flash 写代码
- **法律文书实体提取**：本地大模型 vs 纯算法，大模型在混合语境下更精准
