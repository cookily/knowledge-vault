# Hermes Agent 日报 5月11日-15日

## 一句话总结
社区爆发期：WebUI 本地化启动、商汤免费 API、Clawke 跨平台管理、EchoMind 长期记忆、MiniMax 全模态创作、Team Deploy 团队部署、Scale-Engine 工作流框架等密集开源。

## 核心内容

### 5月11日
- **Hermes WebUI 本地化**：中文社区启动适配，11 个汉化补丁，一键安装提示词上线
- **商汤日日新**：SenseNova 开放免费 API 额度，每 5 小时 1500 次
- **OpenRouter 免费模型**：有调用限制，英伟达官网开放一年免费额度

### 5月12日
- **v0.13 自然语言创建 Agent**：对话中下达指令即可生成独立运行的多 Agent
- **AgentKey**：聚合 Tavily/Brave/Perplexity 搜索 + 微博/微信/小红书/B站/知乎/抖音数据
- **三层架构自进化引擎**：单 Agent 实现，档案袋方案应对长上下文截断

### 5月13日
- **Hermes Team Deploy**：轻量级多用户部署方案，单服务器 + SSH 通道实现权限隔离
- **Scale-Engine**：支持 16 个 Agent 平台的工作流框架（Gitee 托管）
- **Codex Skill Universe**：Skill 管理与自动推荐

### 5月14日
- **供应链安全警告**：PyPI 包 mistralai 被植入恶意代码，需检查依赖版本
- **EchoMind Memory Skill**：跨框架长期记忆（六类记忆 + 反馈强化 + SQLite）
- **MiniMax 全模态创作技能**：飞书内生成图片/视频/语音/音乐

### 5月15日
- **Clawke**：跨平台 AI Agent 管理工作区（Mac/Win/Linux/iOS/Android），CUP 协议
- **EchoMind Memory Skill**（再次推荐）：零外部依赖，跨会话保留代码风格
- **UniPet**：通用 AI Agent 桌面宠物
