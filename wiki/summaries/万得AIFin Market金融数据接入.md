# 万得 AIFin Market：给 AI 接上金融数据能力

## 一句话总结
万得通过 MCP 协议开放金融数据和投研 Skill 给 AI Agent，一句指令即可接入 A 股/港股/美股行情、财报、估值模型等专业数据。

## 核心内容
- **AIFin Market**：万得给 AI 时代的能力平台，提供三样东西——金融数据、严选 Skill、旗舰 Agent
- **数据范围**：A 股/港股/美股行情与财报、ETF/公募基金、债券、指数、宏观经济、上市公司公告
- **数据来源**：与 Wind 终端同源，不需要 Wind 终端账号，独立注册 aifinmarket.wind.com.cn
- **严选 Skill**：DCF 估值模型、季度财报点评、盘后复盘、个股投资逻辑、巴菲特价值投资框架、信用分析、可比公司分析等
- **旗舰 Agent**：Wind Alice Agent，一句话丢任务自动拆解→取数→跑模型→写结论
- **接入方式**：一句话指令 `阅读 https://aifinmarket.wind.com.cn/skill.md 安装万得金融能力`
- **兼容工具**：WorkBuddy、OpenClaw、Hermes Agent 及任何支持 MCP 的 AI Agent

## 使用流程
1. 访问 aifinmarket.wind.com.cn 注册获取 WIND_API_KEY
2. 在 AI Agent 中输入安装指令
3. 根据 Agent 指引完成配置，开始提问金融问题
