# 摘要：UI-UX-Pro-Max-Skill — 一句话生成完整设计系统

## 一句话总结

UI-UX-Pro-Max-Skill（79.6K stars）不是组件库，是 AI 设计智能引擎——输入一句话描述需求，直接输出包含配色/字体/动效规范/反模式清单的完整设计系统，背后是 161 条行业专属推理规则。

## 核心内容

### 传统 vs UI-UX-Pro-Max-Skill

| | 传统方式 | UI-UX-Pro-Max-Skill |
|--|----------|---------------------|
| 输出 | 组件拼凑 | 完整推理设计系统 |
| 定制化 | 依赖模板 | 161 条行业规则按品类推理 |
| 维护 | 需手动更新 | AI 自动适配 |
| 效率 | 设计师手动推导 | 一句话秒出 |

### 161 条行业推理规则示例

以"美容院"为例：
- 品类判断：wellness → Soft UI Evolution 风格
- 主色：Soft Pink #E8B4B8
- 辅色：Sage Green #A8D5BA
- CTA 色：Gold #D4AF37
- 字体搭配：Cormorant Garamond + Montserrat

**不是泛泛的"科技蓝+无衬线字体"，而是针对具体行业推理出来的。**

### 输出内容

一句「帮我做个美容院的落地页」后，输出：

- 页面结构（版块布局）
- 配色方案（主色/辅色/CTA色/中性色）
- 字体搭配（标题/正文/数字）
- 动效规范（入场动画/交互反馈）
- 反模式清单（什么不该做）
- 预交付检查表

### 技术特点

- **非模板**：是推理引擎，每次生成都是推理结果
- **行业专属**：规则按行业分类（科技/健康/电商/金融等）
- **可验证**：输出包含检查表，便于验证是否符合预期
