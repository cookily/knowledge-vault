1.`nvm` 的配置信息存储在一个名为 `settings.txt` 的文件中：
```
C:\Users\chenyl\AppData\Roaming\nvm
```
2.修改 `root:` 这一行。例如：改为D:\nvm_workspace
- **root**: 指向 nvm 安装目录（存放各个 node 版本的地方）。
- **path**: 指向 nodejs 的快捷方式（符号链接）位置，建议保持默认或设为一个固定的非版本路径。
```
root: D:\nvm_workspace
path: C:\Program Files\nodejs
```
3.复制这些版本到新的位置
