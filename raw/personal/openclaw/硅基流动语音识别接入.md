# 硅基流动语音识别接入 OpenClaw 教程

## 概述

通过环境变量 + OpenClaw 配置，让 OpenClaw 使用硅基流动平台的开源语音识别模型做语音转文字。兼容 OpenAI Whisper API 格式。

## 先决条件

1. 注册/登录 [硅基流动](https://siliconflow.cn/)
2. 获取你的 API Key
3. 确认账户有可用免费额度

## 第一步：配置环境变量

文件位置：`C:\Users\chenyl\.openclaw\.env`

添加内容：
```env
# 硅基流动 API 配置
OPENAI_API_KEY=你的硅基流动API密钥
OPENAI_API_BASE_URL=https://api.siliconflow.cn/v1
WHISPER_MODEL=TeleAI/TeleSpeechASR
```

**说明：**
- `OPENAI_API_KEY` → 替换成你自己的硅基流动 API Key
- `OPENAI_API_BASE_URL` → 固定这么写，转发到硅基流动
- `WHISPER_MODEL` → 默认模型，建议用 `TeleAI/TeleSpeechASR`

## 第二步：配置 OpenClaw 通道

文件位置：`C:\Users\chenyl\.openclaw\openclaw.json`

在 `channels.feishu` 里面添加 ASR 配置：

```json
{
  "channels": {
    "feishu": {
      // ... 其他已有配置保持不变 ...
      "prefer_feishu_asr": false,
      "asr": {
        "provider": "siliconflow",
        "model": "TeleAI/TeleSpeechASR"
      }
    }
  }
}
```

**配置说明：**

| 配置项 | 值 | 说明 |
|--------|-----|------|
| `prefer_feishu_asr` | `false` | ❶ **推荐：始终用硅基流动转写，不受飞书转错影响**<br>`true` 优先用飞书自带转写（免费但可能不准），飞书转错就出错 |
| `asr.provider` | `siliconflow` | **必须加！** 指定 ASR 提供者是硅基流动，否则默认找 OpenAI 会报错 |
| `asr.model` | `TeleAI/TeleSpeechASR` | 使用的硅基流动模型 |

## 第三步：重启生效

修改完配置后，**必须重启 OpenClaw** 才能加载新配置。

## 可选模型（硅基流动免费可用）

| 模型名称 | 说明 | 推荐度 |
|---------|------|--------|
| `TeleAI/TeleSpeechASR` | 中文识别效果较好，免费 | ⭐⭐⭐⭐⭐ **当前在用** |
| `SenseVoiceSmall` | 轻量快速，硅基官方推荐 | ⭐⭐⭐ |
| `FunAudioLLM/SenseVoiceSmall` | SenseVoice 完整版 | ⭐⭐⭐⭐ |
| `iic/SenseVoiceSmall` | 原版 SenseVoice | ⭐⭐⭐ |
| `openai/whisper-large-v3` | OpenAI 大模型，一般需要付费 | ⭐⭐⭐⭐⭐ |

## 常见问题排查

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| `Error when processing media: openai api key is not configured for whisper` | 漏了 `asr.provider: "siliconflow"` | 加上这一行配置就好了 |
| 飞书转写错，我回答错问题 | `prefer_feishu_asr: true` 导致用了飞书错的结果 | 改成 `prefer_feishu_asr: false`，始终用硅基转写 |
| 404 Not Found | 模型名称不正确，硅基上没有这个模型 | 换一个模型名称，确认硅基有这个模型 |
| 401 Unauthorized | API Key 错了 | 检查 `OPENAI_API_KEY` 是否正确替换 |
| 识别不准 | 模型问题或者录音噪音大 | 1. 换推荐的 `TeleAI/TeleSpeechASR`；2. 说话凑近麦克风降低噪音 |

## 切换回飞书自带转写

如果想改回优先用飞书：
```json
"prefer_feishu_asr": true,
"asr": {
  "provider": "siliconflow",
  "model": "TeleAI/TeleSpeechASR"
}
```
- 优点：飞书转好直接用，节省硅基配额，速度快
- 缺点：飞书转错你就会得到错误回答，无法挽回

## 切换回 OpenAI 官方 Whisper

如果以后想切回去：

**环境变量：**
```env
OPENAI_API_KEY=你的OpenAIAPI密钥
OPENAI_API_BASE_URL=https://api.openai.com/v1
WHISPER_MODEL=whisper-1
```

**OpenClaw 配置：**
```json
"asr": {
  "provider": "openai",
  "model": "whisper-1"
}
```

重启生效。
